# 遊び方
1. git cloneする
1. npm installする
1. bower installする
1. public/scripts/main.jsのhostを書き換える
1. $ node server/app.js してwsサーバーを立てる
1. public/index.phpにアクセス

# 必要な環境
- apache, php
- node.js（v0.10.32）
- npm（1.4.28）

# Tips
ローカルマシンの/etc/hostsを編集すれば好きなドメインで見られる

e.g. http://local-tomchat.jp/public/
